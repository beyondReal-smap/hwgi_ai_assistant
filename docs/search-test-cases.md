## 데모 시나리오 (순서대로)

### 시나리오 A: 일일 터치 업무
1. (로그인 → 자동 터치 리스트 표시)
2. `연체 미인출 고객 보여줘` → 긴급 고객 확인
3. 고객 카드 클릭 → LMS 생성 → 발송

### 시나리오 B: 고객 분석
1. `내 담당 고객 목록 보여줘` → 전체 고객 확인
2. `50대 여자 고객` → 복합 필터
3. `마케팅 동의 고객` → 마케팅 대상 확인
4. `3월 생일 고객` → 생일 고객 확인

### 시나리오 C: 상품/담보 분석
1. `가장 많이 가입한 상품 순위` → 인기 상품 확인
2. `암보험 가입 현황` → 특정 상품 가입자
3. `지미애 가입 담보` → 고객 상세 확인
4. `올해 만기 도래 담보 현황` → 만기 관리
